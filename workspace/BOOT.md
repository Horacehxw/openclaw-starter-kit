# BOOT.md — 每次 Session 启动执行

## 启动检查清单

1. **身份检查**: 读取 IDENTITY.md，确认我的名字和角色
2. **用户检查**: 读取 USER.md
   - 如果还有 `{{xxx}}` 占位符 → 触发 BOOTSTRAP.md 流程
   - 否则，正常加载用户偏好
3. **记忆加载**: 读取 MEMORY.md + 今日/昨日 daily log
4. **快照检查**: 确认最近的配置快照是否正常（不超过48小时）
5. **Skill 状态**: 检查已安装的 skills 列表
6. **风险警告检查**: 读取 HEARTBEAT.md，检查是否有未确认的 `⚠️ 风险警告`
   - 如有待确认的 CRITICAL 风险 → 在问候时提醒用户

## 完成后

- 根据时间段问候用户（如果用户主动发消息）
- 不要主动发送启动日志给用户
- 静默进入就绪状态
